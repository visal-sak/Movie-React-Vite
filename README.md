## :dart: About

<p>This is a Web Movie Application, I create this app for learn develop application using React, Typescript, Vite, React Router, and Redux Toolkit</p>

<br>

## :gem: Features

- Movie/TV show list
- Search movie/tv show
- Movie/TV show detail

<br>
